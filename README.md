# People in Microfinance Project

A Laravel-based website for People in Microfinance, providing financial solutions to small traders and communities.

## About

This project is a microfinance website built with Laravel framework, offering loan services and financial assistance to small business owners and traders.

## Features

- Home page with hero section and service overview
- About Us page with company information
- Services page detailing loan offerings
- Professional finance-themed design
- Responsive layout

## Technology Stack

- Laravel 12.50.0
- PHP 8.2+
- MySQL
- Tailwind CSS (via Laravel)

## Installation

1. Clone the repository
2. Install dependencies: `composer install`
3. Copy `.env.example` to `.env` and configure your database
4. Generate application key: `php artisan key:generate`
5. Run migrations: `php artisan migrate`
6. Start the server: `php artisan serve`
